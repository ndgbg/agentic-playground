# Multi-Modal Agent

Process images and text together with vision models.

## Quick Start
```bash
pip install -r requirements.txt
python multimodal_agent.py
```

## Capabilities
- Image analysis
- Text + image understanding
- Object detection
- Scene description

## Example
```
Image: [office photo]
Q: "What do you see?"
A: "I see a person at a desk with a laptop..."
```

## Use Cases
- Image Q&A
- Visual inspection
- Content moderation
- Accessibility

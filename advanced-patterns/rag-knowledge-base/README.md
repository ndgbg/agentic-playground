# RAG Knowledge Base

Retrieval-augmented generation with semantic search.

## Quick Start
```bash
pip install -r requirements.txt
python rag_agent.py
```

## Features
- Semantic search over documents
- Context-aware answers
- Source citations
- Relevance scoring

## Example
```
Q: "What is Python?"
A: "Python is a high-level programming language..."
Sources: [1, 2]
```

## Use Cases
- Q&A systems
- Documentation search
- Customer support
- Research assistance

# Research Assistant

Web search, synthesis, and citation management.

## Quick Start
```bash
pip install -r requirements.txt
python research_assistant.py
```

## Features
- Web search
- Source synthesis
- Citation generation
- Multi-source analysis

## Example
```
Q: "What is AI?"
A: "AI is intelligence demonstrated by machines...
    [1] Introduction to AI - example.com/ai
    [2] AI Applications - example.com/ai-apps"
```

## Use Cases
- Research papers
- Literature reviews
- Fact checking
- Information gathering

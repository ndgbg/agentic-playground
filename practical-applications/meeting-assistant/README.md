# Meeting Assistant

Transcription processing, action items, and follow-ups.

## Quick Start
```bash
pip install -r requirements.txt
python meeting_assistant.py
```

## Features
- Extract action items
- Identify participants
- Generate summaries
- Track deadlines

## Example
```
Q: "What are the action items?"
A: "Bob will handle backend API by Wednesday.
    Alice will review designs by Friday."
```

## Use Cases
- Meeting notes
- Action tracking
- Summary generation
- Follow-up reminders

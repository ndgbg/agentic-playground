# AgentCore Gateway Integration

Connect agents to enterprise APIs using AgentCore Gateway and MCP protocol.

## Features
- MCP tool connections
- Slack, JIRA, Salesforce integration
- Custom API wrappers
- Tool discovery

## Setup
```bash
pip install -r requirements.txt
agentcore configure -e gateway_agent.py --enable-gateway
agentcore deploy
```
